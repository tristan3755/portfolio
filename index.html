Mes compétences

html5

Le HTML5 est le langage de base pour le web. Bien structuré et rédigé, il est la pierre angulaire d'un bon référencement naturel.Une rigueur que j'applique à tous mes projets

css3

Le CSS me permets de mettre en forme votre projet, de le rendre attrayant et ergonomique pour vous.

ES6

J'utilise le javascript afin d'animer mes projets. Il me sert le plus souvent dans la manipulation du DOM. Il est ausi utile pour interroger des API REST via des requêtes AJAX et ainsi créer des applications web.

php

Le PHP est un langage back-end qui permet de rendre dynamique le site via l'interrogation d'une base de données. Je suis capable de réaiser une connexion sécurisée via PDO avec une architecture MVC.

three.js

Cette librairie est l'une des rares librairies que j'utilise, elle permet d'eviter les calculs de vecteur et de matrice du Webgl. Avec three.js il est possible de réaliser facilemment une scène pour accueillir un objet 3D.

blender

Blender est un logiciel de création 3D. Couplé avec three.js, il est possible d'intégrer un objet 3D dans une application web. J'ai effectué le robot ci-dessus via ce logiciel. J'utilise Blender que depuis peu et j'ai donc encore beaucoup à apprendre.

Adobe Illustrator

J'utilise illustrator afin d'améliorer l'aspect visuel. J'ai réalisé mon portrait présent tout au début de ce site via l'outil plume d'illustrator. Mon utilisation la plus importante de ce logiciel est dans la modification de dessin déja existant provenant de Freepik

Adobe PhotoShop

PhotoShop me sert pour la réalisation de mock-up. Il est aussi trés utile pour redimensionner des images trop grandes et donc trop lourdes.

MySQl

Afin de gérer la partie back-end, je me tourne vers mySql avec PhpMyadmin. Je sais réaliser des tables avec relations et cardinalité.Cependnat je souhaite m'orienter vers MongoDb.
